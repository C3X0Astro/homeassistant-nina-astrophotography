## N.I.N.A. Astrophotography

Connect [N.I.N.A.](https://nighttime-imaging.eu) to Home Assistant via the Advanced API plugin.

**Requires:** N.I.N.A. with the [Advanced API plugin](https://github.com/christian-photo/ninaAPI) installed (port 1888).

### What you get
- 22 sensors (temperature, guiding RMS, HFR, RA/Dec, sequence progress...)
- 14 binary sensors (connected, tracking, guiding, dome open...)
- 7 controllable number entities (gain, offset, binning, focuser position...)
- 4 switches, 2 selects, 1 light, 13 buttons
- 20 services for full rig control
- Real-time WebSocket push events
- 4 automation blueprints
- Custom Lovelace observatory card
